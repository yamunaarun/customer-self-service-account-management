# Customer Self-Service Account Management System

This repository contains the source code and project documentation for the **Customer Self-Service Account Management System**.

## Project Description
The Customer Self-Service Account Management System is designed to allow customers to manage their accounts independently.  
It provides features such as viewing account details, updating personal information, checking transaction history, and raising service requests, reducing the need for manual support.

## Key Features
- Customer registration and login
- View and update account details
- Transaction and account history
- Secure access and data management
- User-friendly interface

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Java / Spring Boot
- Database: MySQL
- Tools: Git, GitHub, VS Code

## Purpose
This project is developed as part of an academic/team project to demonstrate practical implementation of software engineering concepts such as version control, database management, and full-stack development.
